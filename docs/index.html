Instalar EstatiCar v1.5

Tutorial de Pré-requisitos

Antes de instalar o EstatiCar, você precisa configurar o ambiente com Docker e Ollama. Siga os passos abaixo:

1. Instalar o Docker

O Docker é essencial para o funcionamento do aplicativo. A forma mais fácil de instalá-lo é através do Docker Desktop.

Baixar o Docker Desktop para Windows →

Siga as instruções de instalação para Windows. Após a instalação, certifique-se de que o Docker Desktop está rodando corretamente.

2. Instalar o Ollama e Modelos de IA

O Ollama permite rodar modelos de linguagem (como Mistral e SmolLM) localmente na sua máquina.

Baixar o Ollama para Windows →

Após instalar o Ollama, abra seu terminal (Prompt de Comando, PowerShell ou Terminal) e execute os seguintes comandos para baixar os modelos necessários: 

ollama run mistral:latest
ollama run smollm2:1.7b

3. Rodar o Docker Compose

O pacote de instalação do EstatiCar (que você baixa no botão "Instalar" acima) incluirá um arquivo chamado docker-compose.yml.

1. Navegue pelo terminal até a pasta onde você extraiu os arquivos de instalação. Padrão: "C:/Program Files (x86)/EstatiCar"
2. Execute o comando abaixo para iniciar o serviço do EstatiCar: 

docker compose up

Após alguns instantes, o aplicativo EstatiCar estará pronto para ser executado.
